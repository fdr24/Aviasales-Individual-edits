# Aviasales-Individual-edits
Индивидуальные правки в проекте по ОП часть 1
# -
Для запуска необходимо запустить manager.py
# -
Об индивидуальных правках: я решил развивать идею нашего проекта, который из себя представляет сервис для покупки билетов. Нами юыли реализованы функции по подбору и поиску билетов, но не была реализована возможность купить этот самый билет. Поэтому мной был написан новый модуль purshase_window.
# -
Список добавлений:
# -
1) Теперь с каждым варинтом билета появилась кнопка "Купить", при нажатии на которую открывается новое окно для покупки билета. 2) Реализовано окно для покупки, где пользователь вводит все нужные данные. 3) Реализована проверка ввода данных через валидатор. 4) Проводится проверка, что все поля заполнены, если это не так, то появляется всплывающие окно, сообщающие об этом пользователю.
